## 1-9 搭建Zipkin服务端

- 搭建Zipkin服务端
    - pom.xml
    - 创建 ZipkinApplication 启动类
    - 创建 application.yml 配置文件
    - 访问 [Zipkin WEB UI](http://localhost:62100/zipkin/)

> 如没有特殊要求可以直接使用官方下载的 Zipkin.jar, 直接运行 java -jar Zipkin.jar
